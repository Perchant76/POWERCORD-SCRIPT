## Linux
Dependencies:
1. [git](https://git-scm.com/downloads)
2. [nodejs](https://nodejs.org)(LTS recommended)

### UBUNTU Installation:
1. open terminal and run:

`git clone https://github.com/Perchant76/POWERCORD-SCRIPT`

`cd POWERCORD-SCRIPT`

`chmod +x install-ubuntu.sh`

`./install-ubuntu.sh`
### ARCH Installation:
1. open terminal and run

`git clone https://github.com/Perchant76/POWERCORD-SCRIPT`

`cd POWERCORD-SCRIPT`

`chmod +x install-arch.sh`

`./install-arch.sh`
